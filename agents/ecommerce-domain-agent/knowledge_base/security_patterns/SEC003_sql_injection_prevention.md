# SEC003: SQL Injection Prevention

**Category**: data_security
**Severity**: critical
**Compliance**: OWASP

## Description

        Prevent SQL injection attacks.

        Prevention:
        - Use parameterized queries
        - Input validation
        - Escape special characters
        - Least privilege DB users
        - WAF rules

        Testing:
        - Automated security scans
        - Penetration testing
        - Code review
